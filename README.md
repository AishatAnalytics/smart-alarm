# Smart Alarm Clock

Weather aware alarm clock that adjusts your wake time based on current conditions.

## What It Does
- Fetches live weather data
- Adjusts wake time based on conditions
- Adds extra time for rain snow or extreme heat
- Sends morning alarm email with weather summary

## Sample Results
Temperature: 64F overcast clouds
Wind: 17 mph
Recommended wake time: 6:00 AM
Adjustment: Normal conditions

## Tech Stack
- Python 3
- OpenWeatherMap API
- AWS SES
- boto3
